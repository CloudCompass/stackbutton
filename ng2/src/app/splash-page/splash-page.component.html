What is StackButton?

Stackbutton is designed for DevOps and delivers an easy-to-deploy, central monitoring system for cloud-based services. It is designed to play with these technologies.

NodeJS

NodeJS - Stackbutton is built in JavaScript and runs on NodeJS. This makes interactions from a browser quick and responsive.

Angular

Angular - Stackbutton is built in Angular and TypeScript, which allows for object-oriented web development and responsive UI.

PatternFly

PatternFly - The user interface is built in PatternFly which is a UI framework for enterprise web applications.

Docker

Docker - Stackbutton is bundled inside of a docker container and can be easily built, shared, and run with a single command.

MongoDB

MongoDB - Data is stored and retrieved using MongoDB, allowing for perky data retrieval.

Openshift

OpenShift - is an application development and hosting platform that automates the provisioning, management, and scaling of Stackbutton alongside other containers.
